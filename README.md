# Real-Time Hand Gesture Recognition

A deep learning system that detects and classifies hand gestures 
in real time using MediaPipe and TensorFlow.

## Tech Stack
Python, TensorFlow, OpenCV, MediaPipe, NumPy, Scikit-learn

## Gestures Supported
fist, open_palm, thumbs_up, peace, pointing

## How to Run
pip install opencv-python mediapipe tensorflow scikit-learn numpy
python gesture.py
